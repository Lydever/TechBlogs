设计流程

追求创新、注重细节、 永不放弃、博弈乐趣、顾客至上、为您着想、让您无悔！

工业设计

ID(Industry Design) 包括手机的外观、材质、手感、颜色配搭，主要界面的实现与及色彩等方面的设计。

结构设计

MD(Mechanical Design) 手机的前壳、后壳、手机的摄像镜头位置的选择，固定的方式等工艺都非常熟识。

硬件设计

HW(Hardware) 硬件主要设计电路以及天线，而HW是要和MD保持经常性的沟通。

软件设计

SW(Software) 手机操作界面的模式，经常看到的手机九官格操作菜单的实现，这都是SW设计的范畴。

项目管理

TPM (Technologly Of Project Management)， 专门管技术的PM，而普通的PM，只管理项目的进度各协调工作.

资源开发

不停地去挖掘新的资源，如新材质、新的手机组件、测试器材等，当手机开始试产时，保证生产线上所需要的所有生产物料齐备。

循序渐进,精益求精

工业设计（ ID）追求的是视觉感光的效果，结构设计（ MD）就是力求将这种效果真实还原的方式, ID 设计确定手机的外形后， MD就来一步步去搭建这个手机内部的所有零配件。硬件主要设计电路以及天线，实现手机的配置需求,电路部分先根据配置参数制作一个放大版的 PCB主板，进行各种调试，方案可行后再浓缩做成手机主板。主板的设计又分单面布局和双面布局，单面布局意味着主板上的所有零件全部排布在一面， 背面没有零件； 双面布局则是两面都有零件， 而双面布局的主板相对单面会厚一点， 但是单 面主板的面积就会比双面大， 用单面还是双面的主板这就要看结构是怎么设计， 所以说硬件 设计（ HW）设计的过程也是跟结构设计（ MD）协商的过程。

不停的迭代更新，开发新功能

迭代更新，不断优化

软件设计（ SW）是一个无底洞的工程，硬件部分的东西可能一次开发定稿就完成了使命，但是软件开 发必须不停的迭代更新，开发新功能、修复 Bug、完善稳定、开发新功能⋯⋯这样一个无限循环的过程。 更多功能的实现还需要软件层面的创意， 软件设计的使命就是让现有的硬件的潜能发挥到极致， 这也是软件设计的伟大之处。
